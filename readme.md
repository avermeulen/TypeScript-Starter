# TypeScript Starter

A TypeScript starter project for NodeJS.

Created like this:

```

mkdir TypeScript-Starter
cd TypeScript-Starter/
tsc --init
ls
atom .
tsc --watch
clear
npm init --y
clear
npm install --save @types/node
touch .gitignore

```

## Set node version using nvm

`nvm alias default v8.1.2`
